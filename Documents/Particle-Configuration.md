# Slick2D Particle Configuration Properties

*Converted from Particle Configuration.docx — properties for particle emitter XML files*

| Property | Description |
|----------|-------------|
| `adjust` | True if particles need to adjust for movement |
| `adjustx` | Amount to adjust on the x axis |
| `adjusty` | Amount to adjust on the y axis |
| `alpha` | Interpolated value for alpha |
| `angularOffset` | The angular offset |
| `colors` | The color range |
| `completed` | True if the system has completed due to a wrap up |
| `emitCount` | Number of particles that will be emitted |
| `endAlpha` | The ending alpha value |
| `gravityFactor` | The factor of gravity to apply |
| `growthFactor` | The growth factor on the particles |
| `imageName` | The name of the image in use |
| `initialDistance` | The initial distance of the particles |
| `initialLife` | The initial life of new pixels |
| `initialSize` | The initial size of new pixels |
| `length` | The length of the effect |
| `name` | The name attribute |
| `scaleY` | Interpolated value for y axis scaling |
| `size` | Interpolated value for size |
| `spawnCount` | How many particles are spawned each time |
| `spawnInterval` | How often spawning happens (range) |
| `speed` | Speed particles fly out |
| `spread` | Spread of the particles (0 to 360) |
| `startAlpha` | The starting alpha value |
| `useAdditive` | True if additive blending mode should be used |
| `useOriented` | True if quads should be oriented based on velocity |
| `usePoints` | Points indicator |
| `velocity` | Interpolated value for velocity |
| `windFactor` | The factor of wind to apply |
| `wrapUp` | True if we're wrapping up |
